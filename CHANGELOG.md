# Changelog

All notable changes to Fypod will be documented in this file.

## [1.0.0] - 2026-03-27

### Added
- Initial release
- Auto-detection on LinkedIn job pages
- Floating modal with "Piano Black" design
- CV upload (PDF support)
- Sumopod API integration
- Match score analysis (0-100%)
- Gap analysis with skill recommendations
- Scam detection and salary estimates
- Interactive essay preparation with 3 Google-level questions
- AI feedback on essay answers
- History tracking (last 15 analyses)
- Copy analysis as text
- Delete individual history items
- Multiple AI model selection (Pro/Elite/Vision/Flash)
- Two-tab interface (Setup/History)
- Connection status indicator

### Design
- Minimalist black theme (#000 with 90% opacity)
- Semi-transparent backdrop blur
- Smooth animations and transitions
- Responsive layout
- Clean typography

### Technical
- Manifest V3
- Chrome Storage API
- Content script injection
- Vanilla JavaScript (no dependencies)
- PDF file handling
- API key management
