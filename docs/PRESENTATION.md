## Slide 5: The Design - "Piano Black"
**Premium aesthetic = Premium perception**

- Minimalist black theme (90% opacity)
- No clutter, no settings menu
- Two tabs only: Setup + History
- Floating modal with backdrop blur
- Feels like a $50/month SaaS, but it's free

**Design Philosophy**: "If Apple made a job search tool"

## Slide 6: The Intelligence - AI Features

**Match Score**: 0-100% compatibility analysis
**Gap Analysis**: Specific missing skills identified
**Scam Detection**: Linguistic pattern analysis for red flags
**Salary Estimate**: Market-based compensation range
**Interactive Quiz**: 3 essay questions with real-time AI feedback

**Example Question**:
> "In this role at PTDI, you'll face high-concurrency data pipelines. How would you design a fault-tolerant system using the tech stack mentioned in the JD?"
