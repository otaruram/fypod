# Career Insights Feature

## Overview

Fypod's unique Career Insights feature provides 5 additional data points that differentiate it from competitors, giving users a comprehensive view of job opportunities beyond basic matching.

## Features

### 1. 📈 Career Growth Potential (1-5 ⭐)
**What it analyzes:**
- Long-term career trajectory
- Promotion opportunities
- Skill development path
- Industry growth trends

**Example Output:**
```
⭐⭐⭐⭐⭐ (5/5)
"Strong growth path with clear progression to senior roles"
```

### 2. 💻 Tech Stack Modernity (1-5 ⭐)
**What it analyzes:**
- How cutting-edge the technologies are
- Industry adoption rates
- Future-proofing of skills
- Innovation potential

**Example Output:**
```
⭐⭐⭐⭐☆ (4/5)
"Modern stack with React, Node.js, and cloud-native architecture"
```

### 3. ⚖️ Work-Life Balance Indicator
**What it analyzes:**
- Job requirements intensity
- Expected working hours
- Flexibility indicators
- Stress level predictions

**Possible Values:**
- Excellent (Green)
- Good (Light Green)
- Moderate (Yellow)
- Challenging (Orange)

**Example Output:**
```
Good
"Based on job requirements"
```

### 4. 🏢 Company Culture Fit
**What it analyzes:**
- Startup vs Corporate environment
- Team structure
- Work style (remote/hybrid/office)
- Company values and vibe

**Example Output:**
```
Startup - Fast-paced, innovative, collaborative environment
```

### 5. 📚 Learning Opportunities
**What it analyzes:**
- New skills you'll acquire
- Technologies you'll master
- Domain knowledge expansion
- Mentorship potential

**Example Output:**
```
"You'll gain expertise in microservices architecture, Kubernetes orchestration, 
and distributed systems design"
```

When downloading analysis as PNG, insights are included in the image.

## Technical Implementation

### API Integration
The insights are generated by the AI model using enhanced prompts that analyze:
- Job description keywords
- Required skills and experience
- Company information
- Industry context
- Tech stack mentioned

### Data Structure
```json
{
  "insights": {
    "careerGrowth": {
      "score": 5,
      "reason": "Strong growth path with clear progression"
    },
    "techModernity": {
      "score": 4,
      "reason": "Modern stack with React and cloud-native"
    },
    "learningOpportunities": "Microservices, Kubernetes, distributed systems",
    "workLifeBalance": "Good",
    "cultureFit": "Startup - Fast-paced, innovative"
  }
}
```

### CSS Classes
- `.insights-grid` - 2-column grid container
- `.insight-card` - Individual insight card
- `.insight-icon` - Emoji icon
- `.insight-label` - Title label
- `.insight-stars` - Star rating display
- `.insight-value` - Text value (for balance)
- `.insight-desc` - Description text
